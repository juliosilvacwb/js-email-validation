# js-email-validation

Lib to email validate

#### Javascript example:
```js
const email = require('js-email-validation');  
console.log(email.isValidEmail("jsanyemail@mail.com"))
``` 
Output:
true

#### Typescript example:
```ts 
import { isEmailValid } from 'js-email-validation'
console.log(isValidEmail("jsanyemail@mail.com")
```

Output:
true
