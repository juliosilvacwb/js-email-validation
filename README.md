# js-email-validate

Lib to email validate

#### Javascript example:
```js
const email = require('js-email-validate');  
console.log(email.isEmailValid("jsanyemail@mail.com"))
``` 
Output:
true

#### Typescript example:
```ts 
import { isEmailValid } from 'js-email-validate'
console.log(isEmailValid("jsanyemail@mail.com")
```

Output:
true
